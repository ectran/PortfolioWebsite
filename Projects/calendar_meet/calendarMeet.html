CALENDAR MEET

JavaScript Node.js React PostgreSQL AWS CI/CD

CalendarMeet is a full-stack calendar scheduling application designed to help users organize and manage events through an interactive web interface. The system integrates a React frontend with a Node.js backend and PostgreSQL database, while cloud services and CI/CD pipelines handle deployment and storage.

KEY FEATURES

- Interactive calendar scheduling interface using FullCalendar.js.
- Event creation, editing, and management system.
- Cloud-based storage for application assets
- Automated build and deployment workflow
