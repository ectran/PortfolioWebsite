BOOPING BEEP

Design Lead • Technical Artist • Animator

Unity C# 2D Endless Runner Game Jam

OVERVIEW

Booping Beep is a fast-paced 2D endless runner where players control a bee navigating increasingly chaotic obstacle patterns while collecting power-ups and chasing high scores.

PROJECT INTENT

Developed in a game jam environment, this project focuses on arcade clarity, responsive controls, and escalating difficulty curves. The goal was to create an immediately readable and replayable gameplay loop with strong visual feedback.

TECHNICAL CHALLENGES

High-Speed Readability

Maintained clear visual communication as game speed and obstacle density increased.

Arcade Feel Tuning

Adjusted movement, timing, and feedback systems to create responsive, “tight” gameplay feel.

Jam-Time Asset Integration

Built and integrated visual assets quickly under strict time constraints while preserving consistency.
