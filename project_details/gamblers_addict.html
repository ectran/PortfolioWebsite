GAMBLER'S ADDICT

Project Lead • Gameplay Programming • Systems Design • Art • UI • Animation

Unity C# 2D Bullet Hell Rougelike Adaptive Systems Gameplay AI Risk/Reward Systems

OVERVIEW

Gambler’s Addict is a 2D bullet-hell roguelike built around adaptive risk/reward systems. Players place bets on their own performance before entering encounters, where difficulty, aggression, and rewards scale dynamically based on both performance and risk taken.

PROJECT INTENT

The project explores how gambling mechanics can be translated into gameplay systems that affect moment-to-moment combat decisions, turning risk assessment into a core survival skill rather than a meta layer.
