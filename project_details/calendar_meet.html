UNITY

OVERVIEW

CalendarMeet is a full-stack scheduling application designed to help users organize and manage events through an interactive calendar interface. The system integrates a React frontend with a Node.js backend and PostgreSQL database, with cloud infrastructure and CI/CD pipelines supporting deployment and persistent data storage.

PROJECT INTENT

The project was built to explore how modern web applications coordinate frontend interaction, backend APIs, and cloud infrastructure into a single cohesive system. The focus was on building a reliable event management workflow that supports real-time user interaction while maintaining data consistency across services.

SYSTEM ARCHITECTURE

The system follows a standard full-stack architecture: React frontend communicates with a Node.js + Express backend API, which interfaces with a PostgreSQL database for persistent event storage. AWS handles deployment and hosting, while GitHub Actions manages CI/CD automation.
